Bucles e Iteración

For

La declaración for crea un bucle con 3 expresiones opcionales:

La expresión 1 se ejecuta (una vez) antes de la ejecución del bloque de código.
La expresión 2 define la condición para ejecutar el bloque de código.
La expresión 3 se ejecuta (cada vez) después de que se haya ejecutado el bloque de código.

Ejemplo:

while

El bucle while recorre un bloque de código siempre que una condición especificada sea verdadera.

Ejemplo:

do while

El bucle do while es una variante del bucle while.
Este bucle ejecutará el bloque de código una vez, antes de verificar si la condición es verdadera, y luego repetirá el bucle mientras la condición sea verdadera.

Definir y llamar funciones

Definir una función en JavaScript significa crear un bloque de código que realiza una tarea específica. Una función puede recibir datos (argumentos) y puede devolver un resultado (valor). Las funciones ayudan a organizar el código y a evitar la repetición.
Y llamar una función significa ejecutar el código que se ha definido en esa función
